Le Cosmonaute — Mais qu’as-tu à t’agiter comme ça?

Le Chat — Je ne m’agite pas, je suis… agile!

Soudain, le Chat saute vers le Cosmonaute! Celui-ci esquive d’un geste peu élégant, ni conventionnel, mais néanmoins efficace. Le Robot, qui passait par-là pour son indexation hebdomadaire, s’enquit de cette agilation.

Le Robot — Serait-ce la mise en ligne du documentaire Web Chauss’ ton univers, sur la relation qu’entretiennent les sportifs à leurs chaussures, qui provoque autant d’excitation?

Le Cosmonaute — Pardi! Toutes les personnes ayant participé à ce projet sont fières du travail accompli… Mais, dites-moi, n’auriez-vous pas un chat dans la gorge?

Le Chat jaillit du Robot d’un bond.

Le Chat — Vous rendez-vous compte? Un projet conduit sur près d’un an, entre les tournages, le montage, le développement du site… etc.

Ce dont je suis le plus heureux, c’est la façon dont nous avons mené le développement et la co-création du design graphique, dans le même temps. Étape par étape, avec, à chaque fois, quelque chose de complètement fonctionnel et cohérent.

Le Robot — Il est vrai que ces étudiants ont produit un documentaire Web de bonne facture avec des ressources pourtant limitées.

Le Cosmonaute — Avez-vous vu, en outre, le projet de constitution d’une collection photographique communautaire, avec le tag Flickr chauss’ mon univers?

Le Chat — Nous avons pris beaucoup d’autres belles photographies, nous aussi. Dommage que nous ne pouvons toutes les publier (dans la mesure où nous devons respecter le droit à l’image des personnes représentées).
